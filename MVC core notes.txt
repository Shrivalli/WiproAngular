MVC with EF core steps:

1. Install 3 packages - efcore, efcore sql, efcore tools - build your project (dotnet restore in pm console)

2. PM console: 
Scaffold-DbContext "Server=(localdb)\MSSQLLocalDB;Database=WiproSampleDb;Trusted_Connection=True;" Microsoft.EntityFrameworkCore.SqlServer -o Models

3. Add a controller by right clicking on the controller folder. - MVC with Entity framework template
Model - courses
Context - wiprosampledbcontext


4.  Program.cs(before u build the services)

 builder.Services.AddDbContext<wiprosampledbcontext>(options =>
       options.UseSqlServer(builder.Configuration.GetConnectionString("DefaultConnection")));


5. In appsettings.json:(Line no 8)
 
"ConnectionStrings": {
      "DefaultConnection": "Server=(localdb)\MSSQLLocalDB;Database=wiprosampedb;Integrated Security=true"
    },
